# MY PORTFOLIO WEBSITE

This is a simple personal portfolio website created using HTML and basic CSS. The website consists of two pages:

- Home Page – Displays a brief introduction and personal information.
- Contact Page – Provides contact details and a form for visitors to get in touch.

# TECHNOLOGIES USED

- HTML5
- CSS3

# PURPOSE

This project was developed to practice the basics of web development using HTML and CSS and to create a simple personal portfolio website.

- Improve the design with responsive layouts.
